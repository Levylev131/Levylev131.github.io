Word Machine Emulator

A word machine is a system that performs a sequence of operations on a stack of integers.

Use the textbox below to type numbers and values that the Word Machine Can Process.

Accepted Values: 25, DUP, POP, +, -

Please seperate your operations by using a comma (, ) like in the examples below:

EXAMPLE-1: 23 DUP 4 POP 7 +, 5 6 + 7 DUP, 56 78 +

EXAMPLE-2: 56 45 DUP + POP -, 87 DUP 45 + 65

Click the Button Below to Get the Results.




To view the process/stack for each operation click F12 to bring up the Developer Tools and go to the "Console" Tab.

To see the code for this project go to the "Sources" Tab in the Dev Tools and click the files called: "index.html" & "main.js".
